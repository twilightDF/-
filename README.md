# GraduationDesign

基于Java Swing的课程设计/毕业设计选题系统

## 功能描述

>提供学生和教师进行课程设计或者毕业设计选题平台，教师角色通过登入平台发布选题信息和录取相关学生，学生用户进行选题提交教师审核，管理员负责基础数据维护。
